# Portfolio Template

React + Framer Motion + Navy/Gold theme.

## Live Demo
> Vercel link coming soon

## Stack
- React 18 + Vite + Framer Motion
- Tailwind CSS (Navy/Gold)
- Lucide React

## Usage
```bash
npm install
npm run dev
```

## Customize
- Name/title: `src/App.jsx` Hero section
- Projects: `projects` array
- Skills: `skills` array
- Images: `public/images/projects/`
